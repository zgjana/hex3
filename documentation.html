Getting Started with ZGJANA CAR HEX EDIT 2

Installation

  1. Download the appropriate version for your system from our download page
  2. Run the installer executable
  3. Follow the on-screen instructions
  4. Launch the application after installation completes

First Run Configuration

When you first run ZGJANA CAR HEX EDIT 2, you'll be prompted to:

  • Select your preferred language
  • Choose a default workspace directory
  • Configure basic editor settings (font, colors, etc.)
  • Register your license if you have one

Basic Workflow

The typical workflow for working with ECU files:

  1. Open your ECU file (File → Open)
  2. Analyze the file structure
  3. Make necessary modifications
  4. Verify checksums
  5. Save your changes
  6. Write the modified file back to the ECU

User Guide

Main Interface Overview

The main interface consists of:

  1. Menu Bar: Access to all application functions
  2. Toolbar: Quick access to common tools
  3. Hex Editor: Primary workspace for byte editing
  4. Structure Panel: Displays file structure and detected maps
  5. Status Bar: Shows current position and file information

Working with ECU Files

Opening Files

To open an ECU file:

  1. Select File → Open from the menu
  2. Navigate to your file location
  3. Select the file and click Open
Saving Files

To save your changes:

  1. Select File → Save to overwrite the original file
  2. Or File → Save As to create a new file

Advanced Features

Map Visualization

To visualize ECU maps:

  1. Right-click in the Structure Panel
  2. Select "Detect Maps" or load a map definition
  3. Double-click on a detected map to visualize it
Checksum Tools

To verify or repair checksums:

  1. Select Tools → Checksum
  2. Choose the appropriate checksum algorithm
  3. Click "Verify" to check or "Repair" to fix

Frequently Asked Questions

What file formats does ZGJANA CAR HEX EDIT 2 support?

The software supports binary files (.bin), Intel HEX (.hex), Motorola S-Record (.srec), and several other common formats used in ECU programming.

Can I use this software with any vehicle?

While ZGJANA CAR HEX EDIT 2 works with many vehicles, compatibility depends on the specific ECU. We provide definitions for common ECUs, and you can create custom definitions for others.

Is there a mobile version available?

Currently, ZGJANA CAR HEX EDIT 2 is only available for Windows desktop. We may develop mobile versions in the future.

How do I update the software?

You can check for updates from the Help menu. If an update is available, you'll be prompted to download and install it.

What's the difference between Standard and Professional editions?

The Professional edition includes advanced features like scripting, plugin support, batch processing, and priority technical support.

API Reference

Scripting API

The Professional edition includes a powerful scripting API that allows you to automate tasks and extend functionality.

Basic Script Structure
// Sample script to modify fuel map values
function main() {
    // Open the active file
    var file = editor.getActiveFile();
    
    // Get the fuel map
    var fuelMap = file.getMap("Fuel");
    
    // Increase all values by 5%
    for (var i = 0; i < fuelMap.rowCount; i++) {
        for (var j = 0; j < fuelMap.colCount; j++) {
            var value = fuelMap.getValue(i, j);
            fuelMap.setValue(i, j, value * 1.05);
        }
    }
    
    // Update checksum
    file.updateChecksum();
    
    // Save changes
    file.save();
}

Plugin API

You can create custom plugins to add new functionality to ZGJANA CAR HEX EDIT 2.

Basic Plugin Structure
// Sample plugin definition
class MyPlugin {
    constructor() {
        this.name = "My Custom Plugin";
        this.version = "1.0";
    }
    
    init(editor) {
        // Register menu items
        editor.addMenuItem("Tools/My Plugin", () => this.run());
    }
    
    run() {
        // Plugin functionality
        editor.showMessage("My Plugin is running!");
    }
}

// Register the plugin
editor.registerPlugin(new MyPlugin());
